## 3.  JDK

JDK：	Java Development Kit （java开发工具包）

JRE：	Java Runtime Environment （java运行环境）



Java不是解释型语言，Java需要先编译成字节码文件，再通过JVM（Java虚拟机）对字节码文件进行下一步的解析，生成机器码，底层操作系统才能识别机器码进行程序的运行



Java为什么先需要编译？

为了实现Java的一大特性     跨平台（一次编译，到处运行）

如何实现跨平台？

通过编译的方式，Java源代码编译成字节码文件，字节码文件分别放到不同的操作系统中，由JVM将他们解析成对应操作系统的机器码

Test.java --> Test.class -->运行



Java源代码编译成字节码的命令是什么？

``` 
javac Test.java
```

Java字节码运行的命令是什么？

``` java
java Test
```





## 4. 变量

为什么要使用变量？变量是为了解决什么问题？

数据地址存储的问题

数据都是在内存中进行存储的，存在一个问题，程序会随机分配内存区域，内存地址是16进制的数据，随机分配，地址很难记住

使用变量就是为了解决内存地址很难记忆的问题，相当于给内存地址取了一个别名，取数据的时候直接通过变量名取即可，不需要再通过内存地址来取数据了

基本数据类型和引用数据类型在内存中的区别

所有的变量都是存储在栈内存中的，无论是基本数据类型还是引用类型，基本数据类型与引用数据类型的区别在于是否使用了堆内存

基本数据类型不需要使用堆内存，直接将变量的值存入栈内存

引用类型需要使用堆内存，对象时创建在堆内存中的，将堆内存的地址引用赋值给变量

### 数据类型的转换

1. 基本数据类型的转换

- 自动转换
- 强制转换

尽量不使用强制类型转换，因为存在数据精度损失的隐患

``` java
double num = 10.5;
int i = (int)num;
```

byte -->short -->int -->long -->float -->double

从小向大转

2. 引用数据类型的转换（继承）

前提是需要转换的数据类型必须有继承的关系

- 自动转换
- 强制转换

```JAVA
public class Test{
    public static void mian(String args){
        Student student = new Steudent();
        Person person = new Person();
        //自动类型转换
        person = student;
        //强制类型转换
        student = (Student)person
    }
}
```

实现类和接口也可以进行类型转换，因为接口和实现类的本质也是父类和子类的继承关系（接口不能进行强制类型转换，因为接口无法实例化）

接口是父类，实现类是子类

接口是由抽象类（一个类中包含抽象方法）演变而来

抽象父类

```java
public abstract class Test1{
    public abstract void test();
}
```

非抽象子类

```java
public class Test2 extends Test1{
    @Override
    public void test(){
        
    }
}
```

抽象父类可以优化成接口

```java
public interface Test1{
    public abstract void tset();
}
```

非抽象子类变成实现类

```java
public class Test2 implements Test1{
    @Override
    public void test(){
        
    }
}
```



接口和抽象类的区别：

- 关键字不同
- 抽象类可以包含非抽象方法，接口全都是抽象方法
- 具体化方式不同，抽象类是继承，接口是实例化
- 抽象类只能单继承，而接口可以多实现

父类中的构造器，在创建子类对象的时候会自动调用，非抽象类，调用构造器来创建对象，创建父类对象，抽象类调用父类构造器，但不会创建对象





## 5. 运算符

### 逻辑运算符

（比较两个值的逻辑关系）只能用于boolean类型的数据运算，判断boolean数据之间的逻辑关系，包括：与、或、非

与：&&（短路与）、&

或：||（短路或）、|

非：！

短路与和短路或的性能更高

A & B	同时运算A和B，对结果进行逻辑与运算

A && B	优先运行A，，如果结果为false,则不需要运算B



### 位运算符

对表达式以二进制为单位进行运算，将数据全部转换成二进制在进行运算

按位与：&	A & B：每位数字一一对应，若都为1，则记为1，否则为0

按位或：|	A | B：每位数字一一对应，只要有1个为1，则记为1，否则为0

按位异或：^	A ^ B：每位数字一一对应，相同则记为1，不同则为0

左移：<<	A<<B：A*2的B次方

右移：>>	A>>B：A除以2的B次方

if-else和switch-case的区别：

1. if-else可以进行等值判断，也可以进行逻辑判断，switch-case只能进行等值判断
2. switch-case代码结构更清晰易懂



4种循环for、while、do-while、foreach（针对集合的遍历）

for()简写	10.fori

循环四要素：

- 初始化循环变量
- 循环条件
- 循环体
- 更新循环变量



```java
public class Test{
    public static void mian(String[] args){
        int i = 0;
        while(i < 100){
            System.out.println(i);
            i++;
        }
    }
}
```

```java
public class Test{
    public static void mian(String[] args){
        int i = 0;
        do{
            System.out.println(i);
            i++;
        }while(i < 100)
    }
}
```

while和do-while的区别：

while需要先判断循环条件，再执行循环体

do-while第一次不需要判断循环条件，直接执行循环体，第二次才判断循环条件



终止循环：

- 循环逻辑结束
- 循环过程中抛出异常
- 使用break关键字
- 使用continue

break会结束整个循环，continue只结束当前循环，继续执行下一次循环
